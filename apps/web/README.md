# apps/web - TeenCare UI (Next.js)

## Overview
This app now contains only the TeenCare UI source (cloned from `templates/teencare-ui-kit-main`) adapted to Next.js App Router.

## Routes
- `/`
- `/child-profile`
- `/ai-copilot`
- `/mentor`
- `/alerts`
- `/settings`

## Run
```bash
npm install
npm run dev
```

## Validate
```bash
npm run lint
npm run typecheck
```
